# Лабораторная работа №3

## Параллельное умножение матриц с использованием MPI

---

## Задание

Модифицировать программу из л/р №1 для параллельной работы по технологии MPI. Провести серию экспериментов с различными размерами матриц (200, 400, 800, 1200, 1600, 2000) и различным количеством процессов (1, 2, 4, 8).

---

## Описание

В ходе выполнения лабораторной работы последовательный алгоритм умножения матриц был адаптирован для параллельного выполнения.

Основные вычисления были распределены между процессами с использованием технологии MPI.

Каждый процесс выполняет вычисления своей части результирующей матрицы.

Для взаимодействия между процессами использовались функции MPI.

Программа выполняет следующие этапы:

* генерация исходных матриц
* распределение данных между процессами
* параллельное умножение матриц
* сбор результатов
* измерение времени выполнения

## Анализ результатов

Анализ полученных данных показывает, что параллельное выполнение с использованием MPI приводит к ускорению вычислений.

Основные наблюдения:

* увеличение числа процессов уменьшает время выполнения
* наиболее заметное ускорение наблюдается при переходе от 1 к 4 процессам
* при 8 процессах ускорение сохраняется, но становится менее выраженным
* для небольших матриц эффект параллелизации слабее

## Вывод

В рамках лабораторной работы была реализована параллельная версия алгоритма умножения матриц с использованием MPI.

В ходе экспериментов установлено, что:

* использование MPI позволяет ускорить вычисления
* эффективность зависит от размера задачи и числа процессов
