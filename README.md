# Параллельное умножение матриц (OpenMP)

## Описание

Проект посвящён исследованию производительности параллельного умножения матриц с использованием OpenMP на языке C++.
Также реализована проверка корректности результатов с помощью Python.

## Используемые технологии

* C++
* OpenMP
* Python (NumPy)

## Алгоритм

Используется классическое умножение матриц:

C[i][j] = Σ A[i][k] * B[k][j]

Параллелизация реализована с помощью директивы:

#pragma omp parallel for collapse(2)

## Эксперименты

Тестирование проводилось для квадратных матриц следующих размеров:

* 200 × 200
* 400 × 400
* 800 × 800
* 1200 × 1200
* 1600 × 1600
* 2000 × 2000

Для каждого размера:

* генерируются случайные матрицы
* выполняется параллельное умножение
* измеряется время выполнения
* результат сохраняется в файл

## Результаты

| Размер | Время (сек) |
| ------ | ----------- |
| 200    | 0.022       |
| 400    | 0.185       |
| 800    | 1.497       |
| 1200   | 6.411       |
| 1600   | 24.899      |
| 2000   | 45.446      |

## Анализ

* Временная сложность алгоритма ~ O(n³)
* С увеличением размера матрицы время резко возрастает
* Параллелизация ускоряет вычисления, но не меняет асимптотику

## Проверка корректности

Для проверки используется Python-скрипт, который сравнивает результат с вычислением через NumPy.

Запуск:

python matrix.py --dir .

Или для одного набора:

python matrix.py --single matrixA_1_200.txt matrixB_1_200.txt matrixC_1_200.txt
