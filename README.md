#Лабораторная работа №4
Параллельное умножение матриц на GPU

## Цель работы

Освоить основы CUDA и реализовать параллельное умножение матриц с последующим анализом производительности.

## Оборудование

Тестирование проводилось на видеокарте NVIDIA GeForce RTX 4060 Laptop GPU
Compute capability: 8.9
Объём глобальной памяти: 8188 MB

## Результаты тестирования

В ходе выполнения лабораторной работы были проведены вычислительные эксперименты для различных размеров матриц и конфигураций блоков CUDA.
Во всех таблицах приведено время выполнения вычислений (в секундах).

## Анализ результатов

С увеличением размера матриц наблюдается закономерный рост времени выполнения.

Основные наблюдения:

* На размере 1024×1024 наилучший результат показывает конфигурация 16x16
* Для больших размеров (2048 и 3072) наиболее эффективной оказывается конфигурация 8x8
* Конфигурация 32x32 демонстрирует наихудшие результаты во всех экспериментах
* При увеличении размера матриц различия между конфигурациями становятся более выраженными

Причины наблюдаемого поведения:

* эффективность зависит от загрузки GPU и количества активных потоков
* слишком большие блоки (32x32) могут приводить к ухудшению производительности из-за ограничений ресурсов
* меньшие блоки (8x8) обеспечивают более гибкое распределение нагрузки на больших данных

## Вывод

Проведённые эксперименты показывают, что эффективность CUDA-реализации существенно зависит от выбора конфигурации блоков.

На малых размерах оптимальной является конфигурация 16x16, однако при увеличении размера матриц более эффективной становится конфигурация 8x8.

Использование CUDA позволяет значительно ускорить умножение матриц за счёт параллельных вычислений, однако для достижения максимальной производительности необходимо учитывать размер задачи и правильно подбирать параметры запуска.
